# Neon-Chomp

A cyberpunk-styled Pac-Man clone built as a single HTML5 file with no external dependencies.

## Play

Open `index.html` in any modern browser.

## Controls

- **Arrow Keys** or **WASD** to move
- **Enter** or **Space** to start / restart
- **Touch D-Pad** appears automatically on mobile devices (bottom-right corner)

## Features

- Canvas-rendered neon cyberpunk visuals (glowing walls, dots, and characters)
- 4 ghosts with distinct AI personalities (Blinky, Pinky, Inky, Clyde)
- Scatter/chase mode cycling
- Power pellets with frightened ghost mode and combo scoring
- Tunnel wrap-around
- Level progression
- Responsive sizing and mobile touch support

## Tech

Single file (`index.html`) containing all HTML, CSS, and JavaScript. No external assets, frameworks, or build steps.
